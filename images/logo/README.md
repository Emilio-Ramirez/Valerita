# Logo Folder

## Instructions

Please add your company logo to this folder with the name `logo.png`

### Requirements:
- **Filename**: Must be named `logo.png`
- **Format**: PNG (with transparent background recommended)
- **Size**: Height of approximately 40px for optimal display in the navbar
- **Aspect ratio**: Maintain your logo's original aspect ratio

### Where the logo appears:
- Navigation bar (top of every page)
- Links back to the home page when clicked

### Alternative formats:
If you prefer to use a different format (JPG, SVG, etc.), update all HTML files:
- Change `images/logo/logo.png` to your filename
- Files to update: index.html, contacto.html, raad-type-s.html, supra.html, raad-rc.html, h-25.html
